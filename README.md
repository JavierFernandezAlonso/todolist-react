# Crear una aplicación ToDo List en React 

## Sobre mí 😊👇
- Estudiante de Desarrollo de Aplicaciones Multiplataforma
- Junior Developer en formación
- Manejo de Java, Php y HTML
- Experiencia con Apache NetBeans, PhpStorm, Eclipse, phpMyAdmin, mySQL, Oracle VirtualBox y Docker.

## Objetivos 🎯

- Desarrollar una aplicación web sencilla utilizando React que permita gestionar una lista de tareas pendientes.
- Practicar los conceptos básicos de React, como componentes, estados, eventos, renderizado de listas y 
manejo de formularios.

## Requisitos de la práctica 📑

La aplicación deberá incluir las siguientes funcionalidades: 

1. Añadir tareas

○ El usuario podrá escribir una tarea en un campo de texto. <br>
○ Al pulsar un botón, la tarea se añadirá a la lista. <br>
○ No se deberán permitir tareas vacías.

2. Mostrar listado de tareas

○ Las tareas añadidas deberán mostrarse en pantalla. <br>
○ Cada tarea deberá aparecer como un elemento independiente. 

3. Marcar tarea como completada

○ Cada tarea tendrá una opción para marcarla como completada. <br>
○ Las tareas completadas deberán diferenciarse visualmente, por ejemplo, usando texto tachado o un color diferente. 

4. Eliminar tareas

○ Cada tarea tendrá un botón para eliminarla de la lista. 

5. Componentes

○ La aplicación deberá estar dividida, como mínimo, en los siguientes componentes: 

■ App <br>
■ TodoForm <br>
■ TodoList <br>
■ TodoItem <br>

6. Estilos

○ La aplicación deberá tener una presentación visual básica. <br>
○ Se valorará que la interfaz sea clara, limpia y responsive.

## Requisitos opcionales ⭐

Se valorará positivamente añadir alguna de estas mejoras: 

● Contador de tareas pendientes. <br>
● Contador de tareas completadas. <br>
● Filtro para ver: <br>

○ Todas las tareas. <br>
○ Solo tareas pendientes. <br>
○ Solo tareas completadas. 

● Guardar las tareas en localStorage. <br>
● Botón para eliminar todas las tareas completadas. <br>
● Edición del texto de una tarea ya creada.
